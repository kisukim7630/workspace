## 고구마 마켓 PRD

### 1. 목표

*   **핵심 목표**: 지역 기반 중고 거래를 활성화하여 사용자 간의 편리한 거래를 지원하고, 합리적인 소비를 촉진한다.
*   **세부 목표**:
    *   사용자들이 쉽고 빠르게 상품을 등록하고 검색할 수 있도록 직관적인 UI/UX를 제공한다.
    *   안전하고 신뢰할 수 있는 거래 환경을 구축한다.
    *   지역 사회 구성원 간의 연결을 강화한다.

### 2. 타겟 사용자

*   **주요 사용자**: 20-30대 직장인, 30-40대 주부, 자취생
*   **사용자 특성**:
    *   중고 거래를 통해 합리적인 소비를 추구하는 사용자
    *   이사/정리 시 불필요한 물건을 판매하고자 하는 사용자
    *   육아용품/유아용품을 저렴하게 구매하거나 판매하고자 하는 사용자
    *   지역 기반의 커뮤니티 활동에 관심 있는 사용자

### 3. 주요 기능

*   **상품 등록 및 검색**:
    *   상품 이미지, 제목, 내용, 가격, 카테고리, 거래 지역 등을 입력하여 상품을 등록하는 기능
    *   키워드, 카테고리, 지역 등을 기반으로 상품을 검색하는 기능
    *   최신 상품, 인기 상품, 가격 순 등으로 정렬하는 기능
*   **채팅 기능**:
    *   판매자와 구매자 간의 실시간 채팅 기능
    *   거래 약속, 가격 협상, 상품 문의 등을 위한 기능
*   **위치 기반 서비스**:
    *   사용자 위치를 기반으로 주변 상품을 보여주는 기능
    *   거래 희망 지역을 설정하는 기능
*   **인기 상품 추천**:
    *   사용자들의 관심도와 구매 데이터를 기반으로 인기 상품을 추천하는 기능
*   **거래 매너 평가**:
    *   거래 후 상대방의 매너를 평가하고 후기를 남기는 기능
    *   매너 평가 점수를 공개하여 신뢰도를 높이는 기능
*   **나눔 기능**:
    *   물품을 무료로 나눔할 수 있는 기능
    *   나눔 물품을 등록하고 검색하는 기능

### 4. 추가 기능 (선택 사항)

*   **관심 키워드 알림**: 사용자가 설정한 키워드에 맞는 상품이 등록되면 알림을 보내는 기능
*   **커뮤니티 게시판**: 지역 주민들이 자유롭게 소통할 수 있는 게시판 기능
*   **광고 배너**: 지역 상점이나 다른 서비스 광고를 보여주는 기능 (수익 창출)

### 5. UI/UX

*   **디자인 컨셉**: 친근하고 사용하기 쉬운 디자인
*   **주요 컬러**: 따뜻하고 편안한 느낌을 주는 컬러 (예: 파스텔톤)
*   **UI 특징**:
    *   직관적인 아이콘과 메뉴 구성
    *   반응형 디자인 (PC, 모바일, 태블릿)
    *   이미지 중심의 상품 목록

### 6. 개발 계획

*   **개발 방법론**: MVP (Minimum Viable Product) 개발 후 점진적인 기능 추가
*   **개발 기간**: 3개월
*   **주요 개발 단계**:
    *   1단계: 상품 등록, 검색, 채팅 기능 개발
    *   2단계: 위치 기반 서비스, 인기 상품 추천 기능 개발
    *   3단계: 거래 매너 평가, 나눔 기능 개발

### 7. 기술 스택

*   **프론트엔드**: Next.js 15, Typescript, Tailwind CSS
*   **백엔드**: Hono.js
*   **데이터베이스**: Supabase
*   **인증**: Supabase Auth

### 8. QA 계획

*   **테스트 환경**: 다양한 기기 및 브라우저 환경에서 테스트
*   **테스트 방법**: 기능 테스트, 사용성 테스트, 성능 테스트
*   **주요 테스트 항목**:

    *   채팅 기능의 안정성 및 속도
    *   위치 기반 서비스의 정확성
    *   UI/UX의 사용성

### 9. 배포 계획

*   **배포 플랫폼**: 웹 (반응형)
*   **배포 시기**: QA 완료 후 즉시 배포

### 10. 유지보수 계획
