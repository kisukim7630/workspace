# 🎨 UI 전용 목업 디자인 가이드

## 1. Product Summary

**고구마 마켓**: 지역 기반 중고 거래 플랫폼

사용자들이 쉽고 빠르게 상품을 등록하고 검색할 수 있는 직관적인 UI/UX를 제공하는 지역 커뮤니티 기반 중고거래 플랫폼입니다.

## 2. Target Users

- **20-30대 직장인**: 합리적인 소비를 추구하는 직장인
- **30-40대 주부**: 육아용품/유아용품 거래에 관심 있는 주부
- **자취생**: 이사/정리 시 불필요한 물건을 판매하고자 하는 자취생

## 3. Design Mood

- **친근함**: 따뜻하고 편안한 느낌의 파스텔톤 컬러
- **사용성**: 직관적이고 사용하기 쉬운 인터페이스
- **신뢰감**: 깔끔하고 정돈된 디자인으로 신뢰감 제공

## 4. Color Palette

T-001에서 정의한 컬러 팔레트를 사용합니다.

- **Primary**: 주황색 계열 (#F97316) - 메인 액션, 버튼, 링크
- **Secondary**: 보라색 계열 (#D946EF) - 특별 기능, 강조
- **Accent**: 노란색 계열 (#D97706) - 경고, 알림
- **Grayscale**: 회색 계열 - 텍스트, 배경, 경계선

## 5. UI Guidelines

### 5.1 직관적인 아이콘과 메뉴 구성
- 명확한 아이콘 사용 (Lucide Icons)
- 일관된 네비게이션 구조
- 명확한 라벨과 툴팁

### 5.2 반응형 디자인
- **모바일**: 1열 그리드, 세로 스크롤 중심
- **태블릿**: 2열 그리드
- **데스크탑**: 3-4열 그리드, 사이드바 포함

### 5.3 이미지 중심 상품 목록
- 상품 이미지를 카드 상단에 크게 표시
- 이미지 비율: 1:1 (정사각형)
- 호버 시 이미지 확대 효과

## 6. User Journey

```
앱 접속 
  → 상품 검색 (키워드/카테고리/지역)
  → 상품 상세 확인
  → 판매자 문의 (채팅)
  → 거래 약속
  → 거래 완료
  → 평가 (리뷰 작성)
```

## 7. Common Components

### 7.1 Header (헤더)
- **위치**: 모든 페이지 상단 고정
- **구성요소**:
  - 로고 (왼쪽)
  - 검색 바 (중앙, 데스크탑)
  - 인증 버튼 (오른쪽)
- **반응형**: 모바일에서는 검색 바 숨김

### 7.2 Footer (푸터)
- **위치**: 모든 페이지 하단
- **구성요소**: 저작권 정보, 링크

### 7.3 Button (버튼)
- **Primary**: `bg-primary-500 text-white hover:bg-primary-600`
- **Secondary**: `bg-gray-200 text-gray-900 hover:bg-gray-300`
- **Outline**: `border border-primary-500 text-primary-600`

### 7.4 Card (카드)
- **상품 카드**: 이미지 + 제목 + 가격 + 위치 + 좋아요
- **정보 카드**: 제목 + 설명 + 액션 버튼

### 7.5 Modal (모달)
- **배경**: 반투명 오버레이
- **내용**: 중앙 정렬, 둥근 모서리
- **닫기**: X 버튼 또는 배경 클릭

### 7.6 Input Field (입력 필드)
- **기본**: `border border-gray-300 rounded-lg px-4 py-2`
- **포커스**: `focus:border-primary-500 focus:ring-2 focus:ring-primary-500`
- **에러**: `border-red-500`

## 9. Page Implementations

### 9.1 홈 (/) - 최신/인기 상품 노출

**Core Purpose**: 사용자가 최신 상품과 인기 상품을 한눈에 볼 수 있도록 제공

**Key Components**:
- 상품 카드 그리드
- 필터 바 (카테고리, 정렬)
- 검색 바 (선택사항)

**Layout Structure**:
```
┌─────────────────────────────────┐
│ Header (고정)                    │
├─────────────────────────────────┤
│ 필터 바                          │
├─────────────────────────────────┤
│ 상품 그리드 (반응형)              │
│ [카드] [카드] [카드] [카드]      │
│ [카드] [카드] [카드] [카드]      │
├─────────────────────────────────┤
│ Footer                          │
└─────────────────────────────────┘
```

**반응형 동작**:
- 모바일: 1열
- 태블릿: 2열
- 데스크탑: 3-4열

### 9.2 검색 (/search) - 키워드·카테고리·지역별 상품 탐색

**Core Purpose**: 사용자가 원하는 상품을 빠르게 찾을 수 있도록 다양한 필터 제공

**Key Components**:
- 검색 바 (키워드 입력)
- 필터 사이드바 (카테고리, 지역)
- 결과 리스트 (상품 카드 그리드)
- 정렬 옵션

**Layout Structure**:
```
┌─────────────────────────────────┐
│ Header (고정)                    │
├──────────┬─────────────────────┤
│ 필터     │ 검색 바               │
│ 사이드바 │ 정렬 옵션             │
│          ├─────────────────────┤
│          │ 상품 그리드           │
│          │ [카드] [카드] [카드] │
│          │ [카드] [카드] [카드] │
├──────────┴─────────────────────┤
│ Footer                          │
└─────────────────────────────────┘
```

**반응형 동작**:
- 모바일: 필터 사이드바 숨김, 하단 시트로 표시
- 태블릿: 필터 사이드바 표시 (2열 그리드)
- 데스크탑: 필터 사이드바 + 메인 콘텐츠 (3열 그리드)

## 10. Layout Components

### 10.1 Grid 레이아웃

**적용 경로**: `/`, `/search`

**구현**:
```tsx
<div className="grid gap-4 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4">
  {/* 상품 카드들 */}
</div>
```

**반응형 동작**:
- 기본 (모바일): 1열
- `sm:` (640px 이상): 2열
- `lg:` (1024px 이상): 3열
- `xl:` (1280px 이상): 4열

### 10.2 필터 사이드바

**적용 경로**: `/search`

**구현**:
```tsx
<aside className="w-full lg:w-64">
  {/* 필터 컨텐츠 */}
</aside>
```

**반응형 동작**:
- 모바일: 전체 너비 (상단 또는 하단 시트)
- 데스크탑: 고정 너비 256px (왼쪽)

## 13. 스타일 가이드

### 13.1 간격 (Spacing)
- 카드 간격: `gap-4` (16px)
- 섹션 간격: `mb-6` 또는 `mb-8`
- 패딩: `p-4` (카드 내부), `p-6` (섹션)

### 13.2 둥근 모서리
- 카드: `rounded-lg` (8px)
- 버튼: `rounded-lg` (8px)
- 입력 필드: `rounded-lg` (8px)

### 13.3 그림자
- 카드: `shadow-sm` (기본), `shadow-lg` (호버)
- 모달: `shadow-2xl`

## 14. 접근성

- 모든 이미지에 `alt` 속성 제공
- 키보드 네비게이션 지원
- 색상 대비 WCAG 2.1 AA 기준 준수
- 포커스 상태 명확히 표시
